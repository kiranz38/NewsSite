****Steps to start the App**************
Author : Ram date:06/03/2019

1)Go to the code folder in Node cmd prompt and 
do the the NPM Install to load the modules if NodeJS and NPM is pre-installed

2)Check whether Port 3500 is free for app hosting. If not goto web pack config in root folder
 and change the Dev-Server port no to the desired location .

3)Inorder to start the back end json server for json content,
from the root folder in Node js cmd and execute the command "json-server --watch db.json"
The existing json-server is hosted at port 3000

For json server install do "npm install -g json-server" in case if not loaded during node module installation
